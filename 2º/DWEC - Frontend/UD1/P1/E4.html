Definiciones varias

  • URL: (Uniform Resource Locator) Es la dirección única utilizada para identificar un recurso específico en la web, como una página o archivo.
  • HTTP: (Hypertext Transfer Protocol) Es el protocolo de comunicación utilizado para transferir información en la web entre un servidor y un navegador.
  • TCP/IP: (Transmission Control Protocol/Internet Protocol) Es el conjunto de protocolos que permite la transmisión de datos a través de redes como Internet, gestionando el enrutamiento y la entrega de los paquetes de información.
  • Internet: Es una red global de computadoras interconectadas que permite la comunicación y el intercambio de datos a través de protocolos como TCP/IP.
  • W3C: (World Wide Web Consortium) Es una organización internacional que desarrolla estándares y directrices para garantizar el crecimiento y la accesibilidad de la web.
  • HTML: (Hypertext Markup Language) Es el lenguaje de marcado utilizado para crear y estructurar el contenido en la web, como texto, imágenes y enlaces.
